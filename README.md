An app to remind you what you have to do
## Getting Started

``` bash
git clone https://github.com/rifqialfaizi/WhatToDo.git
cd WhatToDo
open WhatToDo.workspace # or xed .

- Run (⌘ + R) or
- Test (⌘ + U)
```
